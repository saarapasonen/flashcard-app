# Flashcards

A web application for studying with flashcards. Users can create projects and add flashcards to them. Each flashcard has a front side (question or term) and a back side (answer or definition).

## Features

- Users can create an account and log in
- Users can create study projects and add flashcards to them
- Flashcards can be edited and deleted
- Users can study a project by going through its flashcards one by one 
  and marking each as known or not known yet
- After a study session, the app shows a completion percentage
- Users can continue a session with only the cards they did not know yet
- Users can search for flashcards by keyword
- The user page shows statistics and a list of the user's projects
